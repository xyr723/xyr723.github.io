MA6514 机器学习与数据科学 — 三套试卷总结

课程：MA6514 Machine Learning and Data Science（南洋理工大学 NTU） 来源 PDF：

  • MA6514 2022-2023 Semester 2（April/May 2023）
  • MA6514 2024-2025 Semester 2（April/May 2025）
  • MA6514 2025-2026 Semester 1（November/December 2025）

三份均为 闭卷考试，时间 3 小时，含 4 道必答大题。

一、整体对比

项目2022-2023 S22024-2025 S22025-2026 S1
考试时间April/May 2023April/May 2025November/December 2025
题目数量4 题（共 6 页）4 题（共 5 页）4 题（共 5 页）
应用背景主线智能制造 / Industry 4.0 / 航运集装箱Industry 4.0 MNC 咨询 / 航运集装箱中型智造企业实习 / 制造业预测性维护

三套试卷的共同知识骨架：数据科学基础 → 降维（PCA）→ 监督/非监督学习 → 贝叶斯统计 → 神经网络/强化学习。

二、2022-2023 Semester 2（April/May 2023）

Q1. 智能制造 vs 智能制造（概念题）

  • 描述 Smart Manufacturing 与 Intelligent Manufacturing 并写 ≥100 字短文说明两者是否有区别（10 分）。
  • 列出并解释 Industry 4.0 的 4 项最显著收益 并排序（8 分）。
  • 解释 Big Data 的定义并说明 Smart/Intelligent Manufacturing 为何依赖大数据（2+5 分）。

Q2. PCA 与 LASSO / SVM 综合题

  • 三篇历史散文的「词频矩阵」表格：运用 PCA 分析共同作者（3 分）。
  • 描述 PCA 数据预处理归一化步骤（5 分）。
  • 给定 83% 方差、16% 方差的前两主成分，推断共同作者身份（5 分）。
  • 绘制 Scree Plot（6 分）。
  • 解释词频矩阵清洗的 Python 脚本主体（见代码）。
  • 辨析 Dimensionality Reduction、Pearson 相关系数 vs 非监督降维（5 分）。
  • LASSO 特征选择与降维（5 分）。
  • SVM 核技巧的重要性及其计算降维意义（5 分）。

Q3. SVM 三样本求解 + 贝叶斯回归

  • 给定三点训练样本 $x_1,x_2,x_3$ 与 RBF 核，列目标函数 $L(\alpha)=\sum\alpha_i-\tfrac12\sum\sum\alpha_i\alpha_jy_iy_jk(x_i^Tx_j)$ 约束 $\alpha_i\ge0, \sum\alpha_iy_i=0$。
    • 引入等式约束推导新拉格朗日（用松弛变量 “δ”）。
    • 写出决策边界（10 分）。
  • HPD 区间 vs ROPE 区间 的定义与关系（4 分）。
  • 给出 24 行的 Bayesian 回归 PyMC3 代码，逐行解释含义（12 分）。

Q4. 集装箱航运数据 + 贝叶斯模型 + 神经网络

  • Pandas 加载多表 Excel 数据训练集（6 分）。
  • 写一段清洗函数代码的主体（6 分）。
  • 贝叶斯模型代码（pm.Normal/pm.HalfNormal/pm.Deterministic）实现（6 分）。
  • 对比 Bagging vs Pasting、Variance Reduction vs Overfitting、Bagging vs Voting（6 分）。
  • 图示 2-3-2 的 ANN（2 输入 + 1 隐藏层 3 神经元 + 2 输出），给出 W⁽¹⁾、W⁽²⁾ 权重矩阵：
    • 逐神经元前向计算 $z^{(i)}$；
    • Sigmoid 激活得 $o^{(i)}$；
    • 计算 MSE 损失（已知预测 $\hat{y}=[0.475, 0.505]$）。

三、2024-2025 Semester 2（April/May 2025）

Q1. MNC 数据科学咨询（概念题）

  • 作为高层顾问面对一家新加坡电信 MNC：
    • 列出数据科学/Industry 4.0 的三大核心（5 分）；
    • 简述 数据科学广义三要素（5 分）；
    • 总结 Industry 4.0 五大关键技术 并说明在 MNC 的落地（10 分）；
    • 写一份执行摘要：为什么这五项技术最关键（10 分）。

Q2. PCA + LASSO + SVM + ANN 反向传播

  • 作者鉴定任务：PCA、Scree Plot、特征值分解 vs SVD 差异。
  • Kernel Trick 在 SVM 中的意义（5 分）。
  • 特征降维与 LASSO 的关系（5 分）；
    • 降维适用于其它 ML 算法吗？（5 分）
  • ANN 反向传播：
    • 给出三个偏导链 $\partial L/\partial w^{(i)}$ 与 sigmoid 导数 $\sigma(x)(1-\sigma(x))$；
    • 解释每项的意义（12 分）。
  • K-Means vs EM（期望最大化） 相似性与差异（3 分）；
  • 用 EM 做缺失值填补的关键步骤（4 分）；
  • 汽车属性数据集中 MAR 缺失：应如何处理样本（6 分）。

Q3. 集装箱航运 PyMC 模型代码解释

  • 详解 pm.Normal / pm.HalfNormal / pm.sample(2000, observed=y_3) 24 行代码（12 分）。
  • 解读 Determined 变量 μ、Bayesian Inference、Negative Binomial 的用途与三个参数含义（各 2 分，共 6 分）。
  • 说明 sampling divergence 的出现场景与防范（8 分）。
  • 列举 Regression 中的 两种 Shrinkage 模型（岭、LASSO）的优劣对比（5 分）。

Q4. 自动驾驶 + DQN + SARSA

  • 给出 15 行 DQN 训练伪码（经验回放 + target network + ε-greedy），要求填充 & 解释：
    • 逐行解释关键步骤（初始化、batch sample、target 计算、反向传播）；
    • 自动驾驶 SDC 的传感器 → 动作（变道、避障、启停倒退）映射（9 分）。
  • 给出 SARSA 算法步骤（初始化 Q(i,u)、ε-greedy、探索率衰减），要求：
    • 解释 $Q(s,u)$、$r(i,u)$、$rmm$、$\ell(\cdot)$ 的角色；
    • 讨论 α 的更新策略与理由（9 分）。
  • 对比 DQN vs SARSA 的经验回放差异、E-step 含义等。

四、2025-2026 Semester 1（November/December 2025）

Q1. 中型企业实习场景（概念题）

  • 你在中小型智造企业带一个 Year-2 实习生 6 个月：
    • 向实习生解释 Data Science vs Big Data 的异同（7 分）；
    • 介绍数据分析中 四大挑战（8 分）；
    • 列举数据分析中 最关键的五项趋势 并排序（10 分）。

Q2. 可靠性工程 + PCA 预测性维护

  • 汽车零部件厂 12 台 CNC，每台机器 6 个月的 x/y/z/bearing 温度/主轴 RPM/功率 + machine_state（NORM/WARN/FAIL），部分数据丢失：
    • 使用 Pandas + PCA 填补缺失值（4 分）；
    • 讨论何时该用 Pandas vs PCA（4 分）；
    • 给出伪码：数据预处理 → PCA 降维 → 至少 95% 方差阈值 选主成分数（4 + 5 + 4 分）；
    • 依据代码结构推荐哪些传感器变量最关键用于预测性维护（4 分）。

Q3. 监督 vs 非监督学习 + Mercer 定理 + K-Fold

  • 以一个 ML 算法说明 supervised vs unsupervised 在过拟合/欠拟合上的差异（4 分）；
  • 过拟合检测与解决手段，辅以图示或公式（6 分）。
  • Mercer 定理 的核心内容：
    • 核函数有效性条件（7 分）；
    • 核技巧如何映射到特征空间；
  • K-Fold 交叉验证：
    • 调优/正则化参数选择步骤（4 分）；
    • 回归模型中如何计算交叉验证分数、性能度量选择（4 分）；
    • 泛化能力的解释（8 分）。

Q4. 航运贝叶斯模型 + EM + 强化学习

  • 与 2024-2025 相似的 PyMC Bayesian 代码（Dirichlet + Normal Mixture + StandardScaler + Normal，含 Dirichlet 先验的解释）：
    • 行 19 Dirichlet 的意义；
    • 行 22 Normal Mixture 的贝叶斯公式；
    • 用 Normal Mixture 的动机（7 分）。
  • EM 算法 处理 MAR 数据：
    • E-step、M-step 的数学含义（各 4 分）；
    • 以似然公式 $L(\theta)=\frac1n\sum(y_i-Q^{\text{net}}(s_i,u_i))^2$ 和参数更新 $\theta\leftarrow\theta-\alpha\nabla_\theta L(\theta)$ 为例说明（4 分）。
  • 异常值检测：
    • 列举 2 个判定 outlier 的标准（4 分）；
    • 给出一个 ML 算法检测流程（7 分）。
  • 强化学习设计（迷宫导航）：
    • 解释 RL agent 决策流程（3 分）；
    • 给定 10×10 网格 + 固定墙壁的场景，识别 agent/env/states/actions/rewards（5 分）；
    • 推荐合适算法并说明理由（5 分，Q-learning / SARSA / DQN 对比）。

五、三套试卷共同核心考点

  1. 数据科学基础：数据科学、Big Data、Industry 4.0 的概念界定与管理层沟通式说明。
  2. 降维与特征工程：PCA（含 Scree Plot、95% 方差阈值、特征值分解 vs SVD）、LASSO。
  3. 监督学习：SVM（核技巧、Mercer 定理）、回归（岭、LASSO）、交叉验证（K-Fold）。
  4. 非监督学习：K-Means、EM 算法及其在缺失值填补中的应用、异常值检测。
  5. 贝叶斯建模：PyMC/PyMC3 代码阅读（Normal/HalfNormal/Dirichlet/Mixture）、HPD vs ROPE、先验/后验、采样发散。
  6. 神经网络：多层感知机前向传播、Sigmoid、MSE、反向传播 偏导推导。
  7. 强化学习：MDP 要素、SARSA、DQN（经验回放、target network、ε-greedy）、应用场景识别。
  8. 应用情境：智能制造、航运集装箱、自动驾驶 SDC、预测性维护、迷宫导航。

六、复习建议

  1. 代码阅读能力：三套试卷都考 Python/PyMC 贝叶斯代码逐行解释，务必熟练 pm.Normal / pm.HalfNormal / pm.Dirichlet / pm.NormalMixture / pm.Deterministic / pm.sample。
  2. 概念 + 应用：Industry 4.0/数据科学概念题需结合 具体行业场景（MNC、航运、智造）作答，不能只写定义。
  3. 数学推导：重点练 SVM 对偶目标、ANN 反向传播链式求导、SARSA/DQN 的 Q 更新。
  4. 对比类题目：Bagging vs Pasting / Voting、Pandas vs PCA、K-Means vs EM、Ridge vs LASSO、DQN vs SARSA —— 这是高频考点。
  5. 贴合新版大纲：2024 年起加入 DQN 伪码、MAR/EM 填补；2025 年起加入 Mercer 定理、K-Fold、RL 场景建模，是最新复习重点。
