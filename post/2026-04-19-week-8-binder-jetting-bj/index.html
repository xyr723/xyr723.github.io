3D 打印技术类别：Binder Jetting — 粘结剂喷射 包含：3D Systems ColorJet Printing（CJP，Z Corp 继承）、ExOne ProMetal（金属 / 砂 / 陶瓷）、Voxeljet VX（PMMA / 砂）、Aprecia ZipDose（药物打印）

一、中文总结

1. 基本定义

粘结剂喷射是将液态粘结剂选择性地喷射到粉末床上，使粉末颗粒被粘结成形（green part，生坯），再通过后续固化 + 致密化（烧结 / 渗透 / 浸渍）获得最终零件的 AM 工艺。

与 Material Jetting 的区别：BJ 喷的是"胶"，粉末是建造材料；MJ 喷的是建造材料本身。

2. 可用材料

  • 铸造砂（Foundry sand）
  • 陶瓷（氧化铝、氧化锆等）
  • 金属（不锈钢、工具钢、镍基、钨、铜、青铜、Mg 等）
  • 聚合物（PMMA、石膏 + 彩色墨）
  • 药物粉末（淀粉、多功能辅料 + API）
  • 食品（糖、淀粉）
  • 复合材料

3. 基本过程（七步）

  1. CAD → STL → 切片；
  2. 铺粉：辊子 / 刮刀在构建平台上铺一层粉（50–200 µm）；
  3. 喷胶：喷墨头向设计区域喷射液态粘结剂（溶剂 / 水基 / 光敏）；
  4. 干燥 / 部分固化：电热器扫过粉床，部分干燥胶 + 维持均温；
  5. 平台下降一层，重复 2–4；
  6. 固化（Cure）：把整个粉床放进烘箱加热 → 胶完全干燥 → 生坯具备足够强度 → 吹粉取件；
  7. 致密化（Densification）：
  • 烧结（Sinter）：高温使颗粒相互结合；可到全致密但收缩大；
  • 渗透（Infiltration）：用低熔金属（如青铜）填入开孔，常用于金属件。

4. 关键要素

影响 BJ 质量的四大类因素：

  1. 粉末：形貌 / 球形度、粒径与分布、流动性、堆积密度、润湿性、表面化学；
  2. 粘结剂：喷射性（粘度、挥发）、润湿 / 渗透能力、胶合强度、可燃烧残留；
  3. 工艺参数：层厚、粘结饱和度、喷头清洁频率、固化时间与温度；
  4. 特征设计：小孔、支撑厚度、打印分辨率。

4.1 粘结剂要求

  • 流变学适合喷墨喷射（粘度低、稳定）；
  • 适当润湿 → 能渗透；
  • 足够胶合强度 → 生坯可操作；
  • 烧结时"干净烧掉"，不残留有害化学物。

4.2 液滴—粉末交互

  • 液滴撞击粉床 → 铺展 + 润湿 + 渗透（0.1–1 s）；
  • 渗透多、铺展少 → 特征小（分辨率高）；
  • 铺展多、渗透少 → 特征大（分辨率低）；
  • 这是 BJ 分辨率的关键机理。

5. 3D Systems — ColorJet Printing（CJP）

继承自 Z Corporation（1994 成立，1997 首台 Z402、2000 首台彩色、2005 HD3DP）。

5.1 过程

  1. 铺粉（石膏基复合粉）；
  2. 标准喷墨头喷射彩色粘结剂；
  3. 液滴在粉末上形成"球形聚合体"，毛细力使相邻聚合体合并 → 固态网络；
  4. 粉末堆积密度 50–62%，影响最终力学性能；
  5. 平台下降 → 重复 → 完成 → 吸走多余粉 → 取件 → 浸渍强化剂增加强度。

5.2 优缺点

  • 优点：
    • 高速（喷头一次打印整层）；
    • 多行业通用（汽车、包装、教育、鞋业、医疗、航空）；
    • 操作简单；
    • 材料浪费少（粉可回用）；
    • 全彩 CMYK。
  • 缺点：
    • 力学性能弱于 SLS → 功能性件受限；
    • 表面粗糙，通常需要后处理。

5.3 应用

  • 概念件 / 外观件 / 功能验证件；
  • 运动鞋：Nike、adidas、New Balance、Wolverine 等用 CJP 加速原型；
  • Deckers 全彩打印加速设计验证；
  • Monash 大学用 CJP 打印全彩解剖尸体模型，用于医学教育。

6. ExOne — Metal & Sand Binder Jetting

源自 Extrude Hone，由 Lawrence J. Rhoades 创办；现代 ExOne 提供大尺寸砂 / 金属 BJ 系统。

6.1 典型机型

  • Innovent：160×65×65 mm，120 层 / 小时；
  • S-Max：1800×1000×700 mm，60–85 层 / 小时。

6.2 过程

  1. CAD → 控制单元；
  2. 铺一层钢粉到构建平台；
  3. 工业压电喷墨头喷胶 + 干燥灯快速烘干；
  4. 平台下降 → 重复 → 生坯；
  5. 取出 → 烘箱固化 → 烧结（并烧掉粘结剂）→ 然后 青铜渗透 达到全致密；
  6. 后处理：机加工、抛光、表面涂层。

6.3 材料

SS 316 / 420 / 17-4PH、Inconel 625/718、Fe、Fe-Cr-Al、Cr 铁矿、CoCr、锆英、WC、陶瓷球、硅砂等。

6.4 优缺点

  • 优点：
    • 快（并行打印多件）；
    • 自由度高（内腔、倒扣可做）；
    • 可靠（自调、自校准、自诊断）；
    • 可做大件（1800×1000×700 mm）；
    • 粉末回用率高，材料利用率 ~96%。
  • 缺点：
    • 渗透 / 烧结收缩大、变形风险；
    • 表面粗糙、分辨率 0.5–50 µm（相对 PBF 低）；
    • 多步流程；
    • 金属件常含铜渗透相。

6.5 应用

  • 主流应用：快速制作铸造砂芯 / 砂型（替代硬质工装，小时级出件）；
  • 注塑模具、挤出模具、直接金属件、吹塑模；
  • 旧金属模具修复。

7. Voxeljet — VX 系统

  • 德国 1999 年成立；
  • 材料：PMMA 聚合物粉、铸造砂；
  • 典型应用：熔模铸造（PMMA 件替代蜡模当"消失模")；
  • 优势：可扩展，可做汽车发动机缸体级大件；酚醛砂直接粘结等专利工艺。

8. Aprecia — ZipDose（3D 打印药物）

3D 打印首次获得 FDA 批准的药物：SPRITAM（左乙拉西坦，抗癫痫药）— 2015 年。

8.1 三步过程

  1. 3D 打印：喷墨头将粘结液滴喷到药物粉床 → 铺粉 → 再喷 → 多层形成高孔隙药片；未打印区域的粉末起自然支撑作用（防塌陷）。
  2. 干燥（Drying）：控温 + 湿度 + 时间参数优化 → 固化湿态打印单元。
  3. 收获（Harvesting）：打印件放到有孔的旋转振动盘上，振动使未粘结粉末分离 → 回收 + 与新粉混合循环使用。

8.2 ZipDose 核心优势

  1. 材料灵活：用已获 GRAS 认证的常规药用辅料；
  2. 掩味 / 释放曲线可控；
  3. 剂量灵活：
  • 高剂量 → 把 API 混进粉床；
  • 低剂量 → 把 API 加进粘结液；
  • 单片可达 1000 mg 超高载药量仍能秒速崩解；
  1. 剂型多样：固定剂量组合、缓释、颊下 / 舌下吸收、防伪、超低精确剂量、靶向给药；
  2. 在液体接触时秒级完全崩解 → 儿童 / 老年人 / 吞咽困难患者友好。

8.3 制药 3D 打印的动机

  • 灵活剂型 / 尺寸 / 形状；
  • 活 / 非活成分空间分布可设计；
  • 定制释放曲线；
  • 个性化多药合一（polypill）；
  • 适合儿童 / 老年 / 罕见病；
  • 小批量临床试验快速、便宜；
  • 按需 / 远程生产；
  • 避开过敏原；
  • 反馈快 → 改善医疗质量。

9. 易燃 / 高风险材料的 BJ

  • 室温、常压打印 → 降低可燃金属成形风险；
  • 例如 镁合金（粉 / 片 / 屑接触空气 / 湿气可自燃）用水基低分子醇粘结剂可显著降低打印难度。

10. BJ 的总体局限（Limitations）

  1. 多步工艺 — 必须固化 + 致密化；
  2. 生坯相对密度仅 ~50%（低于 PBF） → 致密化过程中几何变形严重；
  3. 表面粗糙、分辨率 0.5–50 µm（低于 PBF）；
  4. 大多数材料的后处理策略仍在研究中。

二、English Summary

1. Definition

Binder Jetting selectively deposits a liquid binder onto layers of powder to bond particles into a green part, followed by curing and densification (sintering/infiltration).

2. Materials

Foundry sand, ceramics, metal (stainless, tool, nickel-based, W, Cu, Mg), polymer (PMMA, gypsum with coloured ink), pharmaceuticals, food, composites.

3. Process

CAD → slice → spread powder layer → jet binder → dry/partial cure → lower bed → repeat → cure in oven (green) → densify by sintering (and optional infiltration, e.g. with bronze).

4. Key Influencing Factors

Powder (shape, size, flow, packing, wettability); Binder (rheology, wetting, strength, clean burn-off); Process (layer thickness, saturation, cure time/temp); Feature design (pore, support, resolution).

Droplet penetration vs spreading determines resolution: more penetration → smaller features; more spreading → lower resolution.

5. 3D Systems CJP (from Z Corp)

Prints full CMYK colour using standard inkjet heads onto gypsum composite powder. Infiltrated after printing for strength. Pros: fast, versatile, simple, low waste, full colour. Cons: weak mechanically, rough surface. Applications: concept models, sport shoes (Nike, adidas), full-colour anatomy (Monash University cadaver project).

6. ExOne Metal & Sand BJ

Industrial piezo inkjet prints binder onto metal powder; green part sintered (binder burns off) and bronze-infiltrated to full density. Materials: 316/420/17-4PH, Inconel 625/718, Fe-Cr-Al, CoCr, WC, ceramics, silica sand. Systems from Innovent (160×65×65 mm, 120 layers/hr) to S-Max (1800×1000×700 mm). Pros: parallel printing, complex undercuts, ~96% material efficiency, large parts. Primary use: sand cores/molds for casting, injection/extrusion dies, mold repair.

7. Voxeljet VX

PMMA and sand BJ for investment casting (PMMA as lost-pattern). Scalable to very large parts; phenolic direct-binding variants.

8. Aprecia ZipDose (Pharmaceuticals)

Powder-liquid 3D printing licensed from MIT for fast-melt dosage forms. First FDA-approved 3D-printed drug: SPRITAM (levetiracetam) in 2015. Three steps: print → dry → harvest (rotary shaker separates unbound powder, recycled). Benefits: GRAS materials, taste-masking, tunable release, up to 1000 mg dose with seconds-scale disintegration, fixed-dose combos, anti-counterfeit features.

9. BJ for Reactive Materials

Room-temperature, atmospheric process reduces fire/explosion risk for reactive powders such as magnesium (which can spontaneously ignite in air/moisture).

10. Limitations

Multi-step process, green part ~50% density, significant distortion during densification, rougher surface, post-processing not mature for many materials.

三、速记要点 / Quick Recall

  • BJ ≠ MJ：BJ 喷"胶"粘粉；MJ 喷建造材料本身。
  • 七步：CAD → 铺粉 → 喷胶 → 干燥 → 下降 → 烘箱固化 → 烧结 / 渗透。
  • 青铜渗透 是 ExOne 金属件达全致密的经典后处理。
  • CJP（3D Systems） = 唯一的全彩 CMYK 3D 打印（石膏粉 + 彩墨粘结剂）。
  • ZipDose 是首款 FDA 批准的 3D 打印药（SPRITAM，2015）；高孔隙 → 秒级崩解，可载 1000 mg。
  • 液滴渗透 vs 铺展 决定分辨率。
  • BJ 室温常压 → 适合 Mg 等易燃金属。
